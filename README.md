# Slime-Soccer
Слаймы играют в футбол.
📜 ПРАВИЛА
🏆 Игра до 5 голов или до окончания 3 минут.
⚡ Супер-прыжок: удерживай кнопку прыжка дольше — слайм взлетит выше!
💥 Удар головой: отбивай мяч верхней частью тела — удар становится сильнее.
⛔ Центральная сетка: слаймы не могут пересекать середину поля.
